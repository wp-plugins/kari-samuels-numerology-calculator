A Wordpress plugin to convert Names/Places/Dates to Life Path Numbers

== Description ==

This generates an input field via a shortcode that converts names/places/dates to life path numbers.

== Installation ==

1. Upload `Kari-Samuels-Numerology-Calculator-Wordpress-Plugin` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the content editor enter in the shortcode.

== Frequently Asked Questions ==

= How can I enter generate the shortcode =

In the content editor click on the buttons (lion or aquarius) to automatically generate the shortcode.

= Where can I see the calculator =

Go to the page that was generated, and examine the input field. Type in a name/place/date and see a life path number.

= What are the options that the shortcode provides =

The shortcode provides for these options:

show_calculations - Whether to show the calculation to determine the life path number

placeholder_text - what placeholder text the user sees to prompt them

= Can I have multiple shortcodes on a single page =

Yes
